# AI-Driven Student Performance Prediction System (EduPredict)

## 📌 Project Overview
EduPredict is an AI-based web application that predicts student performance using Machine Learning. It helps users estimate academic performance based on various input factors.

## 🚀 Features
- Student Performance Prediction
- User-friendly Streamlit interface
- Machine Learning prediction model
- Student Guide page
- Help page

## 🛠️ Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy
- Joblib

## 📂 Project Structure
```
StudentPerformancePrediction/
│── app.py
│── train_model.py
│── requirements.txt
│── student_data.csv
│── model/
│── images/
│── pages/
```

## ▶️ How to Run the Project

1. Clone the repository:
   ```
   git clone https://github.com/Aarnasingh09/Aarna-Singh-PBEL3.0.git
   ```

2. Open the project folder:
   ```
   cd StudentPerformancePrediction
   ```

3. Install the required libraries:
   ```
   pip install -r requirements.txt
   ```

4. Run the application:
   ```
   streamlit run app.py
   ```

5. Open the local URL shown in the terminal (usually http://localhost:8501).

## 📄 Dataset
The project uses `student_data.csv` for training and prediction.
